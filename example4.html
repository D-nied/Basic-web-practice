인터넷의 동작원리


인터넷이 동작하려면 최소 2대의 컴퓨터가 필요함.
그래서 팀 버너스리는 이후 2개의 프로그램을 개발
1. 웹브라우저
2. 웹서버
웹서버가 설치된 컴퓨터에는 특정한 주소를 부여 + 컴퓨터 내 디렉토리에 index.html이라는 파일을 저장.
다음에는 웹브라우저 설치된 컴퓨터 주소창에 특정 주소 입력 후 엔터를 누르면...

1)웹브라우저 설치된 컴퓨터에서 인터넷을 통해 전기적인 신호를 웹서버 컴퓨터에게 보냄.
-전기 신호 내용: 나는 index.html이라는 파일의 코드를 원함.
2)그럼 웹서버 컴이 코드를 보내줌
=> 브라우저 컴에 코드 도착
=> 코드 읽어서 웹페이지를 화면에 출력
결론: 클라이언트는 요청을 서버는 응답을 하는 기능을 가진다.
